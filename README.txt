1. Start the search engine by simply typing 'start'
2. There will be short delay here to load in the necessary auxillary files.
3. You can now set the limit of the number of websites you want to show by 
	typing in 'set_limit <num>'. The default is set to 5 urls
4. Type in your searches and see the results.
5. Enter 'q' or 'quit' to quit the search engine.
